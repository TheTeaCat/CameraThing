# Hardware

This document details how to put together your own CameraThing.

You'll then need to read [FIRMWARE.md](./FIRMWARE.md) to transform it from a pile of buttons and wires into an actual CameraThing.



## Pinout

| Pin  | Usage             |
| ---- | ----------------- |
| 0    | OV7670 VSYNC      |
| 2    | OV7670 SIOD       |
| 4    | SIM800L Power Key |
| 5    | SIM800L Reset Key |
| 12   | Shutter Button    |
| 13   | Indicator Light   |
| 14   | OV7670 HREF       |
| 15   | OV7670 SIOC       |
| 18   | OV7670 PCLK       |
| 19   | OV7670 D7         |
| 21   | OV7670 D5         |
| 22   | OV7670 D3         |
| 23   | OV7670 D1         |
| 25   | OV7670 XCLK       |
| 26   | SIM800L RX        |
| 27   | SIM800L TX        |
| 32   | OV7670 D4         |
| 33   | OV7670 D6         |
| 34   | OV7670 D0         |
| 35   | OV7670 D2         |
